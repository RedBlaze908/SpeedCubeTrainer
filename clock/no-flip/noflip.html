Clock - No Flip

Let's start by saying that to solve the clock without ever rotating it we have to start from a face do some counting and memorize some numbers, at the end we should go back to the starting face and perform the moves we just counted, for the tutorial we would use the black face as the starting face, first let's see what are the steps to memorize the numbers.

M1

The first number we have to memorize will be the sum of two other numbers, so in the initial face from which we start we are going to see how many hours it takes to go from L to U (remember that we also go anti-clockwise many times).

Once we remember the number we turn the cube with y2 and see how much L needs to get to UL, found the amount of hours we add this value with the previous one and we have the first number to memorize that we would use later.

M2

M3

Third number we memorize by simply looking at the difference in hours between L and U, now we are at the quota of three memorized numbers.

None

M4

For the fourth number we would again do a sum, so we turn the cube with y2 and look from R to D the difference in hours, then turn and do R to DR and add them up, so we get the fourth number to remember.

M5

For the last one (the fifth) we stay on the face we came up with first (so the one where we did R to DR) and now we do the difference in hours between D to C and we are done with the five numbers we have to remember.

Now after memorizing it we do y2 to return to the initial face with which we started memorizing numbers.
