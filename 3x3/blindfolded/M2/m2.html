M2 [Edges]

M2 is a faster method for solving the edges blindfolded than the Beginner's Old Pochmann, it uses M2 instead of T Perm. M2 isn't raccomanded for beginners, because the method has some exceptions and can be way harder for Beginners.

Basic principle

  • In the blindfold the cube is resolved using commutators and a buffer position (starting position).
  • In the M2 method, the buffer position for the edges is UF (Up-Front, which the upper adhesive of the piece between up and front).
  • The idea is:
    • Choose a edge to solve.
    • With a move M2 (moving only the central vertical layer) send the edge from the target position to the buffer.
    • Apply a sequence that exchanges the buffer with a piece in the correct position.
    • Go back with the reverse M2 move.

Target management

  • All edges are "brought" to one of the special positions that can be exchanged with buffer with M2.
  • For most cases it is done:
    • Bring the target to DB (Down-Back) using a setup Move.
    • Do M2 move.
    • Return with the reverse setup.

Particular cases

  • There are 4 "special" positions (those in layer M) that cannot be treated like the others and have slightly different sequences:
    • DF (Buffer herself, don't touch it if not to exchange)
    • UF, UB, DB
  • These require personalized cases in order not to ruin orientation.
  • If the number of exchanges of the edges is odd, you will have a case of parity (classic in the blind) and you will have to manage it by reversing two corners at the end.
